Add different favicons to any pages or posts in WordPress.

== Description ==
This simple plugin allows you to add custom favicons to multiple pages and posts.  You are no longer restricted to having one sitewide favicon.

== Installation ==
1. Install plugin
2. Activate.
3. On bottom of posts and pages there is now a section called Favicon.  You can upload your custom favicon to this section.
4. Publish/Update page/post.

== Frequently Asked Questions ==
How do I use this plugin?
Very simple.  Go to the page or post that you would like to add a custom favicon.  Scroll to the bottom and you will find a section called Favicon.  Upload or select your favicon image.  Update or publish your page and that\'s it.
